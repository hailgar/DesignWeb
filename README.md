# KREATIV Design Studio

Landing page React/Vite untuk studio desain grafis KREATIV.

## Struktur

- `src/components/layout` untuk header dan footer.
- `src/components/ui` untuk komponen UI kecil.
- `src/context` untuk state theme/template warna.
- `src/hooks` untuk hook reusable.
- `src/pages` untuk wrapper halaman.
- `src/pages/sections` untuk section landing page.
- `src/styles/globals.css` untuk style global existing.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```
